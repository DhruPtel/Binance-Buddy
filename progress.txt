# Binance Buddy — Progress

## Current State
Phase 5 agent-first execution complete. tsc --noEmit clean. 19/19 tests pass.
Agent now has 12 tools (was 7). Action buttons route through agent chat instead of
browser prompt(). System prompt enforces execute-first behavior. resolveToken shared
in core. PancakeSwap V3 LP support added. check_positions scans all BEP-20 tokens
via GoldRush (not just SAFE_TOKENS). Max tool rounds raised from 5 to 8.

Token→BNB swaps now work for SAFE_TOKENS (CAKE→BNB confirmed on-chain). Guardrails
disabled for demo — all checks (simulation, spending limit, fee reserve, protocol
allowlist) pass unconditionally. Circuit breaker resets on server startup so stale
failure counts don't block the agent between restarts.

ASTER swap fails with TRANSFER_FROM_FAILED — likely a token with transfer restrictions
(tax/fee-on-transfer) or an approval issue for non-SAFE_TOKENS tokens. Needs
investigation if we want to support arbitrary tokens beyond the safe list.

## Open Problems
1. Dune deep research queries use bnb.transactions (raw tx data) which returns basic
   activity metrics but not protocol-specific events (borrow, supply, liquidation).
   Need decoded tables per protocol (e.g. venus_bnb.Comptroller_evt_*) for meaningful
   lending/liquidity/yield analysis. Use Dune MCP searchTables to discover available
   tables per protocol, then update SQL templates accordingly.
2. DeFiLlama yields endpoints (yields.llama.fi/pools, /chart) are marked Pro in docs
   but work on legacy URLs. Could break anytime. GoldRush fallback is wired but
   provides less data (no APY, no IL). Monitor for 403s.
3. APY data from DeFiLlama is unreliable for small pools — now capped at 500% with
   warning badge. bestApy fixed to use apyBase (organic yield) instead of total apy.

## API Strategy (settled)
- Token balances: Multicall3 on-chain (zero API keys needed)
- Prices: CoinGecko free tier
- Tx history: Moralis free tier (MORALIS_API_KEY) → Ankr paid fallback → [] if neither
- Rate limiter: 30k/day cap, 60s cache — in packages/blockchain/src/rate-limiter.ts

## Blockers
- None — wallet scanner works with zero API keys (token balances + BNB price only)
- Add ANTHROPIC_API_KEY (console.anthropic.com) for AI chat to function
- Add MORALIS_API_KEY (moralis.io, free) for tx history + archetype detection
- Add TELEGRAM_BOT_TOKEN (@BotFather) for Telegram commands
- AGENT_WALLET_PRIVATE_KEY auto-generated on first run; fund with BNB before trading

## Known Design Deviations from PRD
- Trenches unlocks at Bloom (500 XP) not Guardian (2000 XP) — deliberate, lowers barrier for demo
- F6 dynamic API-key-driven tools not implemented — tools are static, keys only affect data fetching
- Server holds an agent wallet private key (encrypted AES-256-GCM) — extension still uses MetaMask for user wallet; agent wallet is operational for server-side execution only
